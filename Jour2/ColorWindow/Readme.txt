*** Can select 2 colors in a custom window name ColorToolWindow
*** First color is for painting and second color is for erasing
*** The display is a n * m grid of colors
*** Each row is represented by a box with a random color just like a pixel
*** When you customize the number of rows and columns, the grid will be updated
*** You can fill each boxes with the same color by the Fill All button. The paint color is the one that
will be used
*** You can drag and drop any game objects to the output renderer
*** When you click save to object, the texture represented by the colored grid will be applied to the object you
drag and drop in.
